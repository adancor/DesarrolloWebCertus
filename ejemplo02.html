CURSO DESARROLLO WEB - CLASE 1

La unidad didáctica Desarrollo Web corresponde al programa de Desarrollador Web Front-End, tiene carácter teórico-práctico. A través de ella se busca que el estudiante sea capaz de crear interfaces de usuario, desarrolladas con tecnología web basada en HTML y CSS, utilizando Bootstrap para gestionar proyectos web.
